raw2HDF5

Converts a 3D 8-bit raw file to HDF5.

Usage: raw2HDF5 [input] [output]

Allowed options are:

-h [ --help ] display this message
-i [ --input ] arg Input raw file.
-o [ --output ] arg Output HDF5 filename.
-x [ --x ] arg x extent.
-y [ --y ] arg y extent.
-z [ --z ] arg z extent.

Example:

$ raw2HDF5 -x 128 -y 128 -z 128 -i input.raw -o output.hd
